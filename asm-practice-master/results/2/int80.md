# int 0x80

int 0x80 — це механізм системного виклику в Linux (x86), який дозволяє програмам звертатися до ядра операційної системи.

Він використовується для виконання таких дій:
- вивід даних
- читання вводу
- робота з файлами
- завершення програми

Перед викликом значення записуються в регістри:
- eax — номер системного виклику
- ebx, ecx, edx — аргументи

Після цього виконується:
int 0x80

Сучасною заміною є syscall (у 64-бітних системах).
